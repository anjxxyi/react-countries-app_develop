# react-countries-app_develop
리액트로 만든 나라검색 앱(develop.ver)

## 주요내용
  - 'react-router-dom' : BrowserRouter, Routes, Route
  - DarkMode 지원
  - fetch, API 활용
  - useEffect, useState
  - components ~> props
  - 검색기능(진행중)
